# Myportfolio
🌌 Waresus Sohada Zihad — Portfolio Website

A modern, animated personal portfolio built with pure HTML, CSS, and JavaScript, featuring a luxury navy & gold theme, interactive UI, and custom canvas animations.

✨ Overview

This portfolio showcases my skills as a Creative Developer & Designer, combining:

Clean engineering

Refined UI/UX design

Smooth animations

Interactive elements

Canvas-based particle effects

The goal was to create a portfolio that feels premium, immersive, and modern — not just another static website.

🎨 Design Philosophy

This project focuses on:

Minimal luxury aesthetic (Navy × Gold palette)

Strong typography (Cormorant Garamond + DM Sans)

Micro-interactions & hover effects

Smooth page transitions

Visual hierarchy & balance

🚀 Features

✔ Custom animated cursor ✔ Particle canvas background animation ✔ Smooth page navigation (SPA-style) ✔ Animated skill progress bars ✔ Scroll reveal animations ✔ Fully responsive layout ✔ Modern glassmorphism project cards ✔ Clean contact section with form layout

🛠️ Tech Stack

HTML5

CSS3 (Custom Properties, Animations, Responsive Design)

Vanilla JavaScript

Canvas API

Git & GitHub Pages

No frameworks. No libraries. Built from scratch.
